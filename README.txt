

# Task Manager API - FastAPI + JWT

Una API RESTful desarrollada con **FastAPI** para gestionar tareas, con autenticación segura mediante **JWT**. Incluye CRUD completo, protección de rutas y documentación automática.

---

# Características
- CRUD para tareas (crear, leer, actualizar, eliminar).
- Autenticación y autorización con JWT.
- Base de datos SQLite (fácil de migrar a PostgreSQL).
- Documentación automática con Swagger y Redoc.
- Estructura modular y buenas prácticas.

---

# Tecnologías
- Lenguaje: Python 3.13.9
- Framework: FastAPI
- ORM: SQLAlchemy
- Validación: Pydantic v2
- Seguridad: JWT (python-jose), Passlib (bcrypt)
- Servidor: Uvicorn
